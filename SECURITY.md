# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please follow these responsible disclosure steps:

1. **Do not** create a public issue. Instead, contact us directly.
2. Please email your findings to: [security@yourdomain.com](mailto:security@yourdomain.com).
3. Provide detailed information about the vulnerability, including steps to reproduce if possible.
4. We will acknowledge your report within 72 hours and keep you updated on our progress.

## Security Contact

For any security-related matters, please reach out to: [security@yourdomain.com](mailto:security@yourdomain.com)

Thank you for helping us keep this project and its users safe!
